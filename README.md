# Noteaide

## 项目简介

**Noteaide** 是一个基于人工智能的三维建模智能体项目。用户可以通过自然语言对话的方式，与 AI 助手交互来创建和编辑三维模型。无需专业的 3D 建模技能，只需描述你想要的模型，AI 就能帮你实现。

## 技术栈

- **React** + **TypeScript** + **Vite**
- **Three.js** - 3D 渲染
- **Chakra UI** - UI 组件库
- **Zustand** - 状态管理
- **ESLint** + **Prettier** - 代码质量

## 快速开始

```bash
npm install
npm run dev
```

## 命令

- `npm run dev` - 启动开发服务器
- `npm run build` - 构建生产版本
- `npm run lint` - 代码检查
- `npm run format` - 格式化代码

## 项目结构

```
src/
├── components/      # 组件
├── pages/          # 页面
├── router/         # 路由
├── App.tsx         # 应用入口
```
